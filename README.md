# Drone Flight Advisor PRO

**Ваш надежный помощник для безопасных полетов на дроне!**

## Что это за сервис?

Drone Flight Advisor PRO — это простое веб-приложение, которое помогает пилотам дронов быстро оценить, являются ли погодные условия в выбранной точке подходящими для полетов. Больше не нужно сверяться с несколькими источниками и держать в голове характеристики своего дрона — приложение сделает это за вас!

## Ключевые возможности

*   **Проверка погоды в любой точке мира**: Просто укажите аэропорт рядом с местом полета или выберите точку на карте, и приложение получит актуальные погодные данные.
*   **Обширная база дронов**: Выберите свою модель из большого списка популярных дронов (DJI, Autel, Parrot и другие) или добавьте собственную модель с уникальными характеристиками.
*   **Четкие рекомендации**: Приложение анализирует скорость ветра, порывы, температуру, осадки и другие параметры, сопоставляет их с возможностями вашего дрона и выдает понятный вердикт:
    *   **GO FOR FLIGHT** (Можно лететь)
    *   **CAUTION** (Лети с осторожностью)
    *   **NO GO** (Полеты запрещены)
*   **Работает офлайн**: Благодаря технологии PWA, вы можете пользоваться приложением даже без доступа к интернету (после первого посещения).

## Как пользоваться?

1.  **Выберите ваш дрон** из выпадающего списка. Если вашего дрона нет, вы можете добавить его, нажав на кнопку "Новый дрон".
2.  **Укажите местоположение**:
    *   Начните вводить название города или код аэропорта в поле поиска.
    *   Или нажмите кнопку "Выбрать точку на карте" и кликните по нужному месту.
3.  **Нажмите кнопку "Анализ условий"**.
4.  Получите подробный отчет и рекомендацию по безопасности полета!

**Летайте безопасно и с уверенностью!**
